WEB-CORE provide the basic functionality for building web sites. Provided functionality:

- HTTP requests
- Error handling
- Change detection
- Pipes
- Data storage
- Virtualizing data
- Settings
- Localization
- Date time formating
- Number formatting

Commands:

- Compile build pipeline : npm run compile-build 
- Only build web-core and publish it: npm run package.
- Compile build and web-core and publish it: npm start.
- Run test: npm test.
- Run test with watch: npm run test-watch.
- Linting: npm run lint


For publishing you need to the set environment variable **NPM_TOKEN**.
Steps for setting **NPM_TOKEN** on mac for zsh terminal:
- Go to the home folder
- Open the .zshrc file in text editor
- Add line **export NPM_TOKEN="MY_TOKEN"**
- Where **MY_TOKEN** is token generated by adminastrator for private package
